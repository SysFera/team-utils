- commande pour lister les tickets actuellement attribués à soi
imdoing mine
=> affichage des tickets CP assignés à l'id qui correspond à $USER
imdoing mine pmartinez
=> affichage des tickets CP assignés à l'id qui correspond à pmartinez

- commande pour lister les tickets pour le sprint courant qui ont un OF
imdoing target
=> affichage des tickets CP avec la target actuelle et un OF

- commande pour créer un ticket "fils" d'un ticket avec OF
imdoing create PARENT "SUBJECT" "DESC"
=> création d'un ticket par l'user $USER qui a pour parent le ticket PARENT, avec le sujet SUBJECT et la description DESC, et comme target l'actuelle

- commande pour logger le temps
timestamp - nom - ticket1 - start
timestamp - nom - ticket1 - stop
timestamp - nom - ticket2 - start
timestamp - nom - ticket2 - stop
timestamp - nom - ticket1 - start
timestamp - nom - ticket1 - stop
timestamp - nom2 - ticket1 - start
timestamp - nom2 - ticket1 - stop

$ imdoing start 753
=> création d'un fichier avec comme nom : `date`-$USER-start-753
$ imdoing start 753 pmartinez
=> création d'un fichier avec comme nom : `date`-pmartinez-start-753

$ imdoing stop
=> création d'un fichier avec comme nom : TIMESTAMP-UNAME-STOP-753

$ imdoing start 753
$ imdoing start 754
=> création d'un fichier avec comme nom : `date`-$USER-start-753
=> création d'un fichier avec comme nom : `date`-$USER-stop-753
=> création d'un fichier avec comme nom : `date`-$USER-start-754
